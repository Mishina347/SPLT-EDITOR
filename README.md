# SPLT EDITOR

SPLT EDITORは、マークダウンベースのテキストエディタで、Webアプリケーションとデスクトップアプリケーションの両方で動作します。Tauriフレームワークを使用して、クロスプラットフォーム対応のデスクトップアプリを提供しています。

## 🚀 主要機能

### 📝 テキストエディタ機能

- **Monaco Editor**: 高機能なコードエディタを採用
- **マークダウン対応**: リアルタイムプレビュー付き
- **シンタックスハイライト**: 複数のプログラミング言語に対応
- **自動保存**: 設定可能な自動保存機能
- **カスタマイズ可能**: フォント、色、レイアウトの設定

### 🖥️ マルチプラットフォーム対応

- **Webアプリケーション**: ブラウザで動作
- **デスクトップアプリ**: Windows、macOS、Linux対応
- **PWA対応**: プログレッシブウェブアプリとしてインストール可能

### ⚙️ 設定管理

- **デバイス別設定**: モバイル/デスクトップで最適化された設定
- **設定の永続化**: ローカルストレージまたはTauri設定ファイル
- **動的設定更新**: ウィンドウサイズ変更時の自動設定調整

## 🛠️ 技術スタック

### フロントエンド

- **React 18**: 最新のReactフレームワーク
- **TypeScript**: 型安全性の確保
- **Vite**: 高速なビルドツール
- **Monaco Editor**: コードエディタエンジン

### デスクトップアプリ

- **Tauri**: クロスプラットフォームデスクトップアプリフレームワーク
- **Rust**: バックエンド言語
- **WebView**: ネイティブWebViewを使用

### ビルド・デプロイ

- **GitHub Actions**: CI/CDパイプライン

- **自動リリース**: マルチプラットフォーム向けビルド成果物の自動生成

## 📦 インストール・セットアップ

### 前提条件

- Node.js 20.x以上
- Yarn 1.22.x以上
- Rust (Tauriビルド用)

### 開発環境のセットアップ

```bash
# リポジトリのクローン
git clone <repository-url>
cd my-note-app

# 依存関係のインストール
yarn install

# 開発サーバーの起動
yarn dev
```

### Tauri開発環境

```bash
# Tauri開発サーバーの起動
yarn tauri dev

# プラットフォーム別開発
yarn tauri:dev:linux    # Linux用
yarn tauri:dev:macos    # macOS用
yarn tauri:dev:windows  # Windows用
```

## 🏗️ ビルド・デプロイ

### Webアプリのビルド

```bash
# 本番ビルド
yarn build

# プレビュー
yarn preview
```

### デスクトップアプリのビルド

```bash
# 全プラットフォーム向けビルド
yarn tauri:build

# プラットフォーム別ビルド
yarn tauri:build:linux    # Linux用 (.deb, .AppImage)
yarn tauri:build:macos    # macOS用 (.dmg, .app)
yarn tauri:build:windows  # Windows用 (.msi, .exe)
```

### 環境変数によるカスタマイズ

```bash
# カスタム環境変数でビルド
export VITE_TAURI_PLATFORM=linux
export TAURI_PLATFORM=linux
yarn tauri:build:linux
```

## 🔧 設定

### エディタ設定

- **フォントサイズ**: 調整可能
- **ワードラップ**: カスタマイズ可能
- **背景色**: テーマに応じて変更
- **テキスト色**: 可読性を重視した設定
- **フォントファミリー**: システムフォントまたはカスタムフォント

### 自動保存設定

- **有効/無効**: 自動保存の切り替え
- **間隔**: 保存頻度の調整
- **条件**: 特定の条件下での自動保存

### デバイス別設定

- **モバイル**: タッチ操作に最適化された設定
- **デスクトップ**: キーボード・マウス操作に最適化された設定
- **自動調整**: ウィンドウサイズ変更時の動的設定更新
